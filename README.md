### What I learned

Reinforced how to center a div both horizontally and vertically, like this: 

```css
.card {
  display: flex;
  flex-direction: column;
  position: fixed;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  border: 1px solid;
  border-radius: 15px;
  background-color: var(--white);
  width: 420px;
  padding: 25px;
  box-shadow: 10px 10px;
}
```
